# 🎮 게임잡스 - 한국 게임업계 신입/인턴 채용 대시보드

> 게임업계 취업을 준비하는 분들을 위한 채용 정보 통합 대시보드

## 📌 핵심 기능

**검증된 채용 사이트 링크 제공**

모든 링크는 **원티드(Wanted), 사람인, 잡플래닛, 링크드인, 인디드** 등 실제 작동하는 채용 사이트로 직접 연결됩니다.

⚠️ **참고**: 게임잡(gamejob.co.kr)의 기존 URL 구조가 변경되어 접근이 불가능합니다. 따라서 원티드/사람인 기반으로 전환하였습니다.

## 💡 사용 방법

### 채용 정보 검색
1. **채용사이트 탭** (기본 화면)에서 원하는 카테고리 클릭
2. 원티드/사람인 등 채용 사이트의 검색 결과 페이지가 새 탭에서 열림
3. 실제 채용 공고 확인 및 지원

### 직접 공고 등록 (관리자)
1. 상단 "공고 등록" 버튼 클릭
2. 공고 정보 입력
3. 미리보기 확인 후 등록

## ⚠️ 주의사항

### 정적 웹사이트 한계
- **자동 크롤링 불가**: 서버 사이드 기능이 없어 실시간 데이터 수집 불가
- **게임잡 접근 불가**: URL 구조 변경으로 기존 링크가 작동하지 않음
- **잡코리아 API 필요**: API 신청 필요로 직접 연동 불가
- **대안**: 원티드/사람인 등 검증된 채용 사이트로 직접 연결

### 데이터 신뢰성
- 모든 링크는 원티드, 사람인 등 검증된 사이트로 연결
- 가짜 데이터나 샘플 데이터 없음
- 실제 채용 정보는 각 사이트에서 직접 확인

## 🤖 채용 정보 자동 수집 (Crawler)

이 프로젝트는 주요 채용 사이트의 데이터를 자동으로 수집할 수 있는 크롤러를 포함하고 있습니다.

### 1. 전제 조건
- **API 서버**: 대시보드가 `tables/job_postings` API를 제공하는 서버에서 실행 중이어야 합니다.
- **URL 설정**: `scripts/crawler.py` 또는 `crawler.js` 내의 `API_URL`을 실제 서버 주소로 수정하세요.

### 2. Python 크롤러 사용법 (권장)
- **필요 라이브러리**: `requests`, `beautifulsoup4`
- **실행**:
  ```bash
  pip install requests beautifulsoup4
  python scripts/crawler.py
  ```

### 3. Node.js 크롤러 사용법
- **필요 라이브러리**: `axios`, `cheerio`
- **실행**:
  ```bash
  npm install axios cheerio
  node scripts/crawler.js
  ```

### 4. 수집 대상 사이트
- **게임잡**: 신입 채용 정보
- **사람인**: 게임 카테고리 신입 정보
- **원티드**: 게임 개발 태그 신입 정보
- **잡코리아/잡플래닛**: 주요 키워드 검색 결과

**면책 조항**: 본 사이트의 채용 일정은 참고용이며, 정확한 일정은 각 회사 공식 채용 페이지를 확인해주세요.
